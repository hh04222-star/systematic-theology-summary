# 📖 조직신학자 디지털 아카이브 & 요약본 (Systematic Theology Archive)

본 프로젝트는 교회사 속 주요 조직신학자 25명의 신학적 발자취와 핵심 교의 체계(신론, 기독론, 구원론, 교회/성례론, 종말론)를 체계적으로 요약 및 비교할 수 있도록 설계된 **자료용 마크다운 요약서**이자 **인터랙티브 웹 대시보드**입니다.

---

## 📂 프로젝트 구조 (Project Structure)

```
systematic-theology-summary/
│
├── data/
│   └── theologians.json      # 25명의 조직신학자 상세 데이터베이스 (JSON)
│
├── index.html                # 대시보드 웹 마크업 (시맨틱 구조)
├── index.css                 # 프리미엄 다크 글래스모피즘 디자인 스타일시트
├── app.js                    # 검색, 필터링, 사상 비교 등 대시보드 로직
│
├── theology_summary.md       # [텍스트용] 25명 신학자 요약 단일 마크다운 문서
├── generate_md.py            # JSON 데이터 -> 마크다운 자동 생성 스크립트
└── README.md                 # 프로젝트 설명서 (본 파일)
```

## ✨ 핵심 제공 기능 (Key Features)

### 1. 25명의 핵심 조직신학자 상세 프로필 수록
* **시대별 분류:** 초대 교부(Origen, Athanasius, Augustine 등)부터 중세(Aquinas, Anselm 등), 종교개혁(Luther, Calvin, Arminius 등), 근대(Schleiermacher, Bavinck 등), 그리고 현대(Barth, Bonhoeffer, Tillich, Moltmann, Rahner, James Cone 등)를 포괄합니다.
* **5대 교의(Loci)별 정리:** 신론, 기독론, 구원론, 교회/성례론, 종말론의 세부 사상을 표준 포맷으로 정리하였습니다.
* **부가 정보 제공:** 각 인물의 주요 대표 저작 목록, 신학을 관통하는 핵심 키워드, 가슴을 울리는 대표 명언, 그리고 후세에 미친 역사적 평가를 망라합니다.

### 2. 프리미엄 인터랙티브 웹 대시보드 (index.html)
* **실시간 다차원 검색:** 신학자 이름뿐만 아니라 "칭의", "삼위일체", "무염시태", "유명론" 등 특정 신학 키워드나 활동 지역으로 실시간 검색이 가능합니다.
* **시대별(Era) 퀵 필터:** 관심 있는 역사적 시대를 클릭하여 관련 신학자 카드들만 모아볼 수 있습니다.
* **주제별(Locus) 집중 보기:** 대시보드 상단의 '강조할 신학 주제' 버튼(예: 신론, 구원론)을 클릭하면, 카드에 일반 요약 대신 **해당 교의에 대한 해당 신학자의 구체적인 견해**가 카드 전면에 바로 노출되어 빠른 주제별 학습을 돕습니다.
* **1:1 사상 비교 대시보드 (Comparison Mode):** 카드 하단의 체크박스를 이용해 최대 2명의 신학자를 선택한 후 '비교하기' 버튼을 누르면, 두 인물의 생애 정보부터 5대 교의적 차이점을 좌우 대조표 형식으로 완벽히 정렬하여 보여줍니다. (예: 칼뱅 vs 아르미니우스, 아퀴나스 vs 바르트 등)
* **비주얼 다크 글래스모피즘:** 고풍스럽고 깊이 있는 다크 테마를 기반으로 하며, 시대별로 톤이 맞춰진 컬러 배지와 부드러운 호버 애니메이션, 그리고 신비로운 그라데이션 배경 구체가 세련된 사용자 경험을 선사합니다.

### 3. 자료 소장용 단일 마크다운 문서 (theology_summary.md)
* 웹 화면을 켜지 않고 오프라인으로 읽거나, Obsidian, Notion 등 개인 지식 관리 앱에 통째로 Import할 수 있도록 구조화된 `theology_summary.md` 파일을 제공합니다.
* 문서 상단에는 편리한 내부 링크(Anchor)로 연결된 목차(TOC)가 구성되어 있어 빠르게 원하는 신학자의 사상으로 이동할 수 있습니다.

## 🚀 로컬에서 웹 대시보드 실행하는 방법

본 프로젝트는 순수 Vanilla HTML/CSS/JavaScript로 구현되어 별도의 빌드 과정이 필요하지 않습니다. 다만, `data/theologians.json`을 `fetch` API로 로드하므로 로컬 웹 서버가 필요합니다.

### 방법 1: VS Code Live Server 사용 (가장 간단함)
1. VS Code에서 본 프로젝트 디렉토리(`systematic-theology-summary`)를 엽니다.
2. `index.html` 파일을 우클릭한 후 **"Open with Live Server"**를 클릭하여 실행합니다.

### 방법 2: Python 간이 서버 실행
터미널을 열고 본 프로젝트 디렉토리로 이동한 뒤, 아래 명령어를 실행합니다:
```bash
# Python 3 버전
python -m http.server 8000
```
그 후 브라우저를 열고 `http://localhost:8000`에 접속합니다.

### 방법 3: Node.js (npx) 사용
Node.js가 설치되어 있다면 다음 명령어로 간편히 실행할 수 있습니다:
```bash
npx http-server -p 8000
```
그 후 브라우저를 열고 `http://localhost:8000`에 접속합니다.
